# Robotika
UAS Pemrograman Robotika.

Soal praktik UAS Pemrograman Robotika.

RC Car dengan ukuran kecil.

Specs:
1. MCU = WeMos D1 Mini (ESP8266)
2. Drive motors = DC Motor
3. Motor driver = L293D
4. RGB lights = WS2812B Smart LED Strip RGB
